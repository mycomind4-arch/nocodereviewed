# No Code Empire Revised

This workspace has been organized into source files, planning docs, exports, assets, and raw imports so future work has a clear home.

## Folder Map

| Folder | Purpose |
| --- | --- |
| `src/components/` | React/JSX and TSX UI components, calculators, microsites, and tools. |
| `src/pages/` | Page entry files and `index*.ts` variants. |
| `src/edge-functions/` | Supabase/edge function source files. |
| `scripts/` | Python and Node scripts used to generate specs, docs, or PDFs. |
| `data/` | SQL schemas and structured project data. |
| `docs/guides/` | Implementation and integration instructions. |
| `docs/specs/` | Architecture and build specifications. |
| `docs/plans/` | Planning documents and implementation roadmaps. |
| `docs/prompts/` | Prompt files and generated prompt results. |
| `docs/reports/` | Reviews, gateway reports, and analysis notes. |
| `docs/master-plan/` | Larger master-plan bundles and related source documents. |
| `exports/html/` | Generated or delivered HTML files. |
| `exports/pdf/` | Generated or delivered PDF files. |
| `exports/docx/` | Generated or delivered Word documents. |
| `assets/images/` | Image assets, screenshots, generated visuals, and bitmaps. |
| `assets/lightburn/` | LightBurn project assets. |
| `archives/` | Zip archives and compressed handoff bundles. |
| `imports/` | Raw extracted/imported files kept for provenance. |
| `install-media/` | App installers and package files. These can usually be moved outside the project. |
| `off-project-media/` | Media that appears unrelated to the No Code Empire project. Review before deleting or moving elsewhere. |

## Notes

- Duplicate-looking files such as `(1)` variants were preserved instead of deleted.
- Root-level `.DS_Store` files were removed.
- No source contents were edited during the organization pass.
