# Lovable Microsite Evidence Upgrade

Static evidence-safe Lovable flagship microsite for NoCodeReviewed.

## Preview

```bash
cd lovable_microsite_evidence_upgrade
python3 -m http.server 8080
```

Open: http://localhost:8080/tools/lovable/

## Status values

- contentStatus: needs-evidence
- evidenceStatus: pending-hands-on-testing
- pricingStatus: needs-verification

## Included

- 13 static HTML pages under `/tools/lovable/`
- `/data/lovable-evidence.json`
- evidence brief and support docs in `/docs/content/lovable/`
- Claude/Codex conversion prompt in `/docs/prompts/`
