# VibeCode Authority Audience And Conversion Model

## Principle

Every reader path should route toward the most useful evidence surface before asking for a commercial action.

The site should convert by reducing uncertainty, not by hiding risk.

## Core Segments

| Segment | Primary Intent | Best Path |
| --- | --- | --- |
| Nontechnical founder | Choose the fastest credible MVP builder | Best tools for founders |
| Technical founder | Move from prototype to maintainable code | AI IDE Transition Report |
| Operator or internal-tools buyer | Build workflows, portals, and admin tools safely | Internal tools guide |
| Agency or consultant | Pick repeatable delivery stacks | Prompt-to-App Benchmark Report |
| AI tooling researcher | Track category movement and evidence quality | Scoring methodology |

## Conversion Rules

| Surface | Goal | CTA | Proof Required |
| --- | --- | --- | --- |
| Homepage hero | Route users by intent | Best tools by use case | Evidence-strip and benchmark language before sales asks |
| Review page | Convert research into tool choice | Product or affiliate link | Disclosure, freshness, artifact, benchmark, and QA gates |
| Best-of page | Capture buying intent | Open ranked review | Gate status and provisional labels for incomplete tools |
| Report page | Build institutional authority | Capture evidence or view readiness | Dated evidence windows and claim-ledger compliance |
| Matcher | Help uncertain users choose a route | Recommended platform review | Frame as current fit until gates pass |

## Operating Rule

Do not optimize for clicks before trust.

The correct sequence is:

1. Segment the reader.
2. Route them to the right evidence surface.
3. Show the evidence state.
4. Explain the next best action.
5. Present monetized CTAs only after disclosure and evidence gates are satisfied.

## Blockers

A conversion module should not publish when:

- It recommends a winner without comparable evidence.
- It places an affiliate CTA near stale pricing.
- It hides missing benchmark or QA status.
- It routes all readers to the same tool regardless of intent.
- It uses urgency language without evidence.
