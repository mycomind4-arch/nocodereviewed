## 6) Autonomy Notes

Adalo should be classified as an **AI-assisted visual no-code app builder**, not a fully autonomous development agent. Official evidence supports AI app generation across Adalo plans and database-driven web/native app creation. It does not prove that Adalo autonomously designs secure data models, handles permissions correctly, or passes app-store review without user work.
