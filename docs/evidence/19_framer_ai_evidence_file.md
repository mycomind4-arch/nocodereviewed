## 6) Autonomy Notes

Framer AI should be categorized as **AI-assisted website creation**, not autonomous software development. Official evidence supports website layout generation, Wireframer, starter content, and design acceleration. It does not support claims that Framer AI autonomously creates secure, compliant, conversion-optimized production websites without human review.

| Autonomy question | Current evidence answer |
