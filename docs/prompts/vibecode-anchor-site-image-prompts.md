# VibeCode Authority Anchor Site Image Prompt Library

Use this as the image-generation prompt library for the anchor site. The visual system should communicate: independent testing lab, AI app-builder category authority, benchmark evidence, production readiness, and practical founder decision-making.

## Global Visual Direction

Apply this style block to most site images unless a prompt says otherwise:

**Style block:** editorial technology photography, premium software research lab, real computer screens and interface fragments, neutral off-white workspace, black and warm graphite UI surfaces, small orange accent details, subtle blue status lights, crisp shadows, high-detail product journalism aesthetic, modern founder/operator audience, realistic materials, no fantasy elements, no abstract gradient blobs, no cartoon mascots, no fake brand logos, no readable copyrighted UI, no distorted text, no clutter.

**Aspect ratios:**

- Hero: `16:9` or `21:9`
- Article header: `16:9`
- Review card thumbnail: `4:3`
- OG/social: `1.91:1`
- Square social: `1:1`
- Tool comparison graphic: `3:2`
- Methodology diagram background: `16:9`

**Negative prompt:** cartoon, mascot, cyberpunk, purple neon overload, unreadable text, fake logos, warped screens, hands with extra fingers, busy collage, generic stock photo, fantasy aura, mystical imagery, crypto aesthetic, bokeh blobs, floating orbs, exaggerated gradients, low-resolution, watermark.
