What you'll have when done

A fully wired Supabase database with 22 tables, RLS policies, and 25 affiliate programs pre-seeded. Vercel-deployed across 10 sites with environment variables, error handling, GDPR compliance, and shared webhook security. Every site talks to the same database.
Content machine
An n8n pipeline that runs every 2 hours — pulling from a prioritized keyword queue, scraping competitors, calling Gemini with per-site voice fingerprints, running a 12-check quality gateway, and publishing automatically. Target: 40 articles/day across the network.
10 live sites

The gaps the prompts don't cover

No prompt covers: the tools database initial population
Prompt #29 gives you a SQL insert for 100 tools. But that data goes stale on day one. The Tool Data Refresh agent (prompt #23) keeps it current — but you need to verify the seed data is accurate before it goes live on review pages. Pricing errors on launch are an HCU risk.
No prompt covers: the n8n instance itself
The pipeline spec assumes a running n8n instance. The prompts don't cover hosting it (n8n Cloud vs self-hosted on Railway/Render), configuring credentials, or the shared Supabase connection across all workflow nodes. Budget 2–3 hours for n8n setup before touching the workflow imports.
No prompt covers: the site_voices table
The n8n pipeline fetches voice fingerprints from a site_voices Supabase table, but no prompt creates this table or seeds it. Prompt #6 gives you the persona content — you need to manually create the table and insert the 10 personas before the pipeline can run.
No prompt covers: Webflow CMS webhook receiver
The /api/publish-article endpoint is specified but the Webflow-specific CMS API implementation (creating CMS items via Webflow's API, not just a generic webhook) isn't covered. Sites built on Webflow need a thin Next.js/Edge Function layer that translates the publish payload into Webflow CMS API calls.
No prompt covers: inter-site linking domain config
Agent 3's cross-site linking requires all 10 domains to be registered and live. The prompts assume this. In practice you need all 10 domains purchased, pointed at Vercel, and SSL-configured before Agent 3 can build the link map — otherwise it's linking to 404s.

The one number to watch

Quality gateway pass rate
If your gateway hard-fail rate is above 30%, you're spending more on Gemini retries than on successful articles. Run a calibration batch of 20 articles before going to full velocity. Tune the voice fingerprint few-shot examples based on what actually passes. This single number determines whether the pipeline runs at $0.03/article or $0.09/article.
Realistic month 12 scenario
If you hit a 70% gateway pass rate, run 3 sites in parallel in months 1–3 instead of all 10, and validate affiliate links before scaling — you reach $35k–55k/month by month 12 rather than $79k. That's still a $1.5–2.5M exit at 40x. The aggressive scenario requires the expired domain to work and all 10 sites live by month 3. Plan for the conservative scenario and treat the optimistic one as upside.
