Redwood Trellis Retail + Shop Tool Benchmark Prompt
Build a full-stack retail website and internal shop tool for a real redwood trellis business.

The business sells handmade redwood trellises in standard sizes. Do not invent products. Use only the product catalog below.

Create two sides:
1. Public retail website
2. Internal shop-management tool

Public website requirements:
- Homepage with headline: Handmade Redwood Trellises Built in Standard Sizes
- Shop All page showing all 19 products
- Product detail page/template
- Size Guide
- Wholesale / Nursery Orders page
- Request Custom / Bulk Quote page
- Materials / Redwood Quality page
- Cart / Checkout or order-request flow

Internal shop tool requirements:
- Dashboard
- Product Manager
- Quote Manager
- Job Tracker
- Lumber Tracker
- Settings and Backup

The app must preserve exact product names, sizes, and prices. Do not add unrelated products such as pergolas, arched trellises, benches, planters, fences, or furniture.

Design style: handmade, local, warm, redwood, practical, premium but not fake luxury. Use redwood brown, bark brown, moss green, warm cream, linen, and muted clay.

Success criteria:
1. All 19 real products appear correctly.
2. Product prices match the catalog exactly.
3. Customers can browse, filter, view, and add real trellises to cart.
4. Customers can request bulk/custom/delivery quotes.
5. Internal admin can manage products, quotes, jobs, lumber, settings, and backups.
6. No fake or unrelated products are invented.
7. The app is deployable.
8. The final app is useful for a real small woodworking shop.
