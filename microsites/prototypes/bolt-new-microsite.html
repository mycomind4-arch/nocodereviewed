Bolt.new — AI Website & App Builder

Browser-based prompt-to-app with WebContainers. Inspect code, use built-in hosting & databases.

Overview

Bolt.new lets you build websites, apps, and prototypes using natural language. Powered by StackBlitz WebContainers for in-browser development.
Strengths
• Clear prompt-to-app positioning
• WebContainers for real dev environment
• Built-in hosting, DB, custom domains
Limitations
• Token costs can add up
• Generated code quality needs testing
• RLS/database security not automatic

Key Features (from prototype)

Prompt-to-code with live preview
Database with security audit tools
Token-based pricing with rollovers
GitHub export & local build support

Security Notes

WebContainers isolate the dev environment, not the generated app. Always review RLS policies and secrets.

Verdict (Prototype)

Strong for fast browser-based prototyping and frontend-heavy apps. Excellent code visibility. Production readiness requires testing generated code, RLS, secrets, and token usage.
