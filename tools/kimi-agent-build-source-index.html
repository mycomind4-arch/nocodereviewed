Reference material only (per AGENTS.md & SOURCE_OF_TRUTH_MAP.md). This is a Vite + React 18 + TypeScript + Tailwind + shadcn/ui + heavy animation (GSAP, R3F, Framer, Lenis) reference app generated as a modern frontend for a no-code review platform. Do not merge directly into production. Useful for UI patterns, component ideas, mock data structure, page layouts, and future migration planning when the nucleus is stable. Production base remains the static vanilla app.

Key Takeaways for Integration (reference only):
• Mock data structure (src/data/mockData.ts) is excellent — detailed Tool schema with ratings, pros/cons, pricingTiers, reviewBody. Could inform evidence-backed data model or static-app tools[].
• Page structure & UX patterns: Tools list with filters/search, Compare, ToolDetail, Blog/Article, Submit — good inspiration for main app sections without copying React.
• shadcn/ui primitives: 40+ accessible components. For future if we ever move to a component system (not now).
• Heavy animation stack (GSAP + R3F + Lenis + Framer) — reference for what "premium" could feel like; keep simple for current static nucleus.
• Public images: High-quality tool and guide visuals — already useful.
• State management (Zustand) and routing (HashRouter) — interesting for future SPA evolution.
